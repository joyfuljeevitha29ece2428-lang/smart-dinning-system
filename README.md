# Smart Dining System

## Project Description
The Smart Dining System is a web application that allows users to browse food items, view menus, and place orders through a simple and user-friendly interface.

## Features
- View restaurant menu
- Search food items
- Add items to cart
- Place food orders
- Responsive design

## Technologies Used
- HTML
- CSS
- TypeScript
- Vite

## How to Run
1. Install dependencies:
   npm install

2. Start the project:
   npm run dev
